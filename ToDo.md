Soll-Anforderungen
===
- [ ] Nutzer kann mit "Lagerbestand aufüllen" ein Standort mit Getränken aus dem Zentrallager auffüllen
- [ ] Nutzer kann einzelne Kästen nachbestellen
- [ ] Alle Getränkearten sollen gleich behandelt werden
- [ ] Verschiedene standorte haben verschiedene Lagerkapazitäten
- [ ] Zum Start ist das Zentrallager gefüllt
- [ ] nutzer kann bestimmte Menge an bestimmten Kästen an anderen Ort verschieben
- [ ] Nutzer kann an jedem Standort Kästen als "verkauft" angeben
- [ ] angebrochene Kästen können nicht verschoben werden
- [ ] Nutzer kann "automatische Nachbestellung" machen, ( Summe aller vorräte ?== Soll, wenn nicht wird nachbestellt)
- [ ] Übersicht aller Kästen, Standorte und zusätzlich Details (Kastenattribute)
